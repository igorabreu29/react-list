# react-list

Projeto desenvolvido através dos desafios do FrontEndMentor

- Projeto desenvolvido com React.js e TypeScript
- Nesse projeto foi desenvolvido uma To-Do List em que podemos ver as tarefas incompletas, as completas e todas juntas.
- Poder altenar entre o tema escuro e claro!
- Todos os dados são persistidos no local-storage
