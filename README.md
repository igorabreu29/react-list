# react-list

Projeto desenvolvido através dos desafios do FrontEndMentor

- Projeto desenvolvido com React.js e TypeScript
- Nesse projeto foi desenvolvido uma To-Do List em que podemos ver as tarefas completas, incompletas e todas juntas.
- Possui a funcionalidade de altenar entre o tema dark e light!
- Todos os dados são persistidos no local-storage
- Para manipulação do formulário foi usado react-hook-form.
